# Features

PlayModeSaver is packed with tools to bridge the gap between Play Mode and Edit Mode.

## Core Features at a Glance

*   **[Auto-Save & Restore](./auto-save.html)**: Never lose a value change again.
*   **[Visual Highlighting](./visual-highlighting.html)**: See exactly what you changed (Blue) and what you added (Green).
*   **[Presets & Snapshots](./presets-and-snapshots.html)**: Save your runtime state as a reusable preset.
*   **[Attributes API](./attributes-api.html)**: Developers can enforce saving with specialized attributes.
